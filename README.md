# Who Do You Call? TED Talk Mindmap

A static rehearsal dashboard for the TED talk script in `ted talk (1).pdf`.

The app includes:

- An interactive mindmap of the talk structure.
- Slide-by-slide memory anchors and key lines.
- Speaker beats for each section.
- A practice timer.
- Focus mode for rehearsal prompts.

Built as plain HTML, CSS, and JavaScript so it can run directly on GitHub Pages.
